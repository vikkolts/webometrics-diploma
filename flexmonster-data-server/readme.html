Thank you for downloading Flexmonster Data Server!

Flexmonster Data Server is a powerful tool for large datasets processing and server-side pivoting. Since the data summarizing is made on the server, Flexmonster Pivot gets the data in a ready-to-show format and visualizes it faster.

Our diagram shows how the Data Server cooperates with Flexmonster Pivot

See advantages of the Data Server

  • Larger datasets. The Data Server can process files more than 1GB — the maximum file size depends on your RAM capacity and the number of unique members in the dataset.
  • Built-in security configurations. Flexmonster Data Server is provided with utilities for establishing authorized access to data and security management.
  • A convenient way of connecting to a database. The Data Server allows connecting to many popular SQL databases in a few lines of code.
  • Cross-platform solution. Flexmonster Data Server is available for Windows, Ubuntu/Linux, and macOS.

Check out supported data sources

The Data Server supports the following data sources:

  • CSV and JSON files
  • MySQL databases
  • Microsoft SQL Server databases
  • PostgreSQL databases
  • Oracle databases
  • Microsoft Azure SQL databases
